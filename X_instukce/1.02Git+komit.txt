    Налаштувати Git
    Зробити перший коміт
    Викласти код на GitHub, 
    щоб уже з першого дня була історія змін і резервна копія.

    1. Ініціалізувати Git у папці проєкту
У терміналі (переконайся, що ти в project_CMR):


git init
Це створить приховану папку .git для відстеження версій.

    2. Додати всі файли

git add .
Крапка означає «додати всі зміни в поточній папці».

    3. Зробити перший коміт

git commit -m "Старт проєкту CRM"
